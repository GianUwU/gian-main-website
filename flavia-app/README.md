# Flavia App

A React application built with Vite and TypeScript for the Flavia service (flavia.gian.ink).

## Getting Started

### Install Dependencies
```bash
npm install
```

### Development
```bash
npm run dev
```

The app will be available at `http://localhost:5176`

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```

### Lint
```bash
npm run lint
```
