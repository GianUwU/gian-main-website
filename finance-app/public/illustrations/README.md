# Illustrations Folder

Add your PNG images here for the finance app empty states!

## Supported Images:

- `empty-transactions.png` - Shown when there are no transactions in the list

## Recommended Specifications:

- Format: PNG with transparency
- Size: 200x200px to 400x400px
- Style: Light, friendly illustrations that match your app's aesthetic
- Color palette: Consider using your app's colors (#F38375, #2A9D8F, #E9C46A, etc.)

## Fallback:

If no image is found, a chart emoji (📊) will be displayed as a fallback.
